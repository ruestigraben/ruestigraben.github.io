Rüstigraben

Bridging the gap between Rust and Java

Our goal

To provide tooling infrastructure for running Rust code on GraalVM.

The name

The name Rüstigraben is derived from the Swiss word Röstigraben (pronounced [ˈrøːʃtiˌɡrabə]). In Swiss culture, it refers to the border between the German- and French-speaking parts of Switzerland. More abstractly, it also represents their cultural differences. Same for Rust (or LLVM) and Java.
